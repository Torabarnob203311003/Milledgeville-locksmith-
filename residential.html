Rekey vs. Replace

Rekey Service & Lock Changes

A lot of customers are unsure whether they should rekey the locks or replace them completely. In many situations, the existing hardware still works properly and only the internal key configuration needs to be changed.

Rekey service changes the lock so old keys no longer work while allowing the existing hardware to remain in place. Proper inspection helps determine which option makes the most sense.
